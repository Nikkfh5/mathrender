# Changelog

## 0.1.0 — 2026-04-01

Initial release.

- VS Code WebView panel with KaTeX + marked.js rendering
- Embedded HTTP server for Claude Code hook communication
- Real-time formula rendering via Stop hook
- Session history with search and copy
- Pause/Resume/Clear controls
- Dark and Light theme support (auto-detects VS Code theme)
- Cross-platform: Windows, macOS, Linux
